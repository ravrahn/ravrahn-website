Websites

Mekong

In late 2013, I was given an assignment to create a simple book shopping website using Perl or Python CGI. I took this as a design exercise as well as a computing assignment, and wrote every line of CSS from scratch. This was the first website I properly designed. I had created a primitive version of ravrahn.com based on Wordpress, but I had no real experience with web design and learnt most of the CSS I used as I wrote it.

All of the code, the assignment spec, and more detail about my solution, are available on the project's Github repo. A fully functional live version is available here.

CSESoc

In early 2014, I volunteered to redesign the website of CSESoc, my university's computer science society. It was being rewritten from the ground up, with a Mezzanine and Django backend, and they wanted a visual refresh. This was my first time writing Javascript.

Given that CSESoc is based around computing, I wanted the CSESoc website to look professional, and dynamic. The header, rather than transitioning from one state to another after a point, shifts continuously as you scroll - this is an effect I really like, but I've never seen it done before.
